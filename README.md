# Nathalie Flusin - Garden Consulting Services

A beautiful, modern portfolio website for Nathalie Flusin's garden consulting services.

## Features

- Responsive design with Tailwind CSS
- React Router for seamless navigation
- TypeScript for type safety
- Modern UI with custom color palette
- Portfolio showcase
- Contact form
- Services overview

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
